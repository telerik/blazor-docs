# Rendering Changes in 14.0.0

@[template](/_contentTemplates/common/general-info.md#compatible-theme)

## SegmentedControl Integration

The SegmentedControl is now integrated in and replaces the existing toggle functionality of the following components:
	
* Scheduler&mdash;The toolbar button group for switching between Day, Week, Month, Timeline, Year, and Agenda views (k-scheduler-views).
* DateTimePicker&mdash;The toggle inside the datetime popup for switching between the Date and Time panels (k-datetime-buttongroup).
* ImageEditor&mdash;The Portrait/Landscape toggle in the crop image action pane.
* Gantt&mdash;Day/Week/Month selector.
