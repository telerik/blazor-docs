# Breaking Changes in 14.0.0

## Chat

The `OnLoadMoreMessages` event arguments now contain the full rendered message range, not just the newly requested page. 

## Menu

The `MenuHideEvent.Click` value of the Menu's `HideOn` parameter now also causes the child items popup to close, when the user clicks or taps a leaf Menu item. 

## PDFViewer

The PDF.js library used internally by the PDFViewer is updated to version 5.x.

## SegmentedControl Integration

The SegmentedControl is now integrated in and replaces the existing toggle functionality of the following components:
	
* Scheduler&mdash;The toolbar button group for switching between Day, Week, Month, Timeline, Year, and Agenda views (k-scheduler-views).
* DateTimePicker&mdash;The toggle inside the datetime popup for switching between the Date and Time panels (k-datetime-buttongroup).
* ImageEditor&mdash;The Portrait/Landscape toggle in the crop image action pane.
* Gantt&mdash;Day/Week/Month selector.

## TabStrip 

The `ActiveTabIndexChanged` event and the `ActiveTabIndex` parameter are now deprecated. Use the `ActiveTabId` parameter with the `ActiveTabIdChanged` event instead.

## ThemeColor Changes

The `ThemeColor.Dark` and `ThemeColor.Light` values of the `ThemeColor` parameter are removed from the following components: 

* Badge
* Button
* DropDownButton
* SplitButton

The `ThemeColor` parameter is removed from:

* Card
* Dialog
* Window
* Popup edit settings

`OverlayThemeColor` is removed from the LoaderContainer. 

The appearance of the components listed above is determined by the active theme style sheet.
